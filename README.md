
Introduction
-
This project aims to provide PySide6 bindings for QCustomPlot and has been tested on MSYS2/MinGW64.

License
-
This program is licensed under the LGPL.

Install & Test
-
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ python ../example/demo.py
```
